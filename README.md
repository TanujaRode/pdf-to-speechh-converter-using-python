# pdf-to-speechh-converter-using-python
This project is a PDF to Speech Converter developed using Python, designed to improve accessibility and user convenience by converting text from PDF documents into spoken audio.
The system extracts text from PDF files using the PyPDF2 library and converts it into speech using gTTS (Google Text-to-Speech). A simple file selection interface is implemented using Tkinter, allowing users to easily choose PDF files from their system. The generated audio is saved as an MP3 file and can be played automatically.
